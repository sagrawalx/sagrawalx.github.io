I'm currently teaching a class on algebraic geometry using Cox, Little, and O'Shea's *Ideals, Varieties, and Algorithms* {% cite clo %}. There is a sequence of results there that go by the name of "extension theorems," as well a closely related sequence of results called the "closure theorem." Here is a scheme-theoretic discussion of these results. 

# Generalities

Suppose $A$ is a noetherian $\mathbb{N}$-graded ring which is finitely generated in degree 1 over $A_0$, and let $\pi : \mathrm{Proj}\, A \to \mathrm{Spec}\, A_0$ be the structure map. 

Every closed subscheme of $\mathrm{Proj}\, A$ is of the form $V_+(I)$ for some graded ideal $I$ of $A$ {% stacks 0BXK %}. Since $\pi$ is universally closed {% stacks 01MF %}, the scheme-theoretic image of $V_+(I)$ under $\pi$ is a closed subscheme of $\mathrm{Spec}\, A_0$ whose underlying set is $\pi(V_+(I))$. We want to explicitly describe the ideal $J \subseteq A_0$ which defines this closed subscheme in terms of $I$. 

**Definition.** If $I$ is a graded ideal in $A$, we define the *saturation* $I^s$ of $I$ by 
$$ I^s = \{ a \in A : a A_n \subset I \text{ for some } n \geq 0 \}. $$
Clearly $I \subseteq I^s$, and we say that $I$ is *saturated* if $I = I^s$. 

**Lemma 1.** Let $I$ be a graded ideal in $A$. Then $I^s = (I : A_+^\infty)$.

*Proof.* Recall that $(I : A_+^\infty)$ is defined by $$(I : A_+^\infty) = \{ a \in A : a A_+^n \subseteq I \text{ for some } n \geq 0 \}.$$
Fix a list of elements $f = (f_0, \dotsc, f_n)$ of $A_1$ which generate $A$ as an algebra over $A_0$. Then $A_+$ is generated as an ideal by $f$, so $A_+^n$ is generated by $f^\alpha$ where $|\alpha| = n$. In other words, $A_+^n$ is generated as an ideal by $A_n$. It follows that $aA_n \subseteq I$ if and only if $aA_+^n \subseteq I$. 

**Proposition 2.** Let $I$ be a graded ideal in $A$. Then $$\pi(V_+(I)) = V(I^s \cap A_0).$$
More precisely, $V(I^s \cap A_0)$ is the scheme-theoretic image of $V_+(I)$ under $\pi$ and its underlying set is $\pi(V_+(I))$. 

*Proof*. Since $V_+(I) = V_+(I^s)$ as closed subschemes of $\mathrm{Proj}\,A$ {% stacks 0BXK %}, we might as well assume that $I = I^s$. We have already noted that the set underlying the scheme-theoretic image of $V_+(I)$ under $\pi$ must be $\pi(V_+(I))$, so it suffices to identify the scheme-theoretic image. 

Note that $V_+(I) = \mathrm{Proj}\, A/I$ and the map $i^\sharp : \mathscr{O}_{\mathrm{Proj}\, A} \to i_*\mathscr{O}_{V_+(I)}$ corresponding to the closed embedding $i : V_+(I) \hookrightarrow \mathrm{Proj}\, A$ is the image of the graded ring homomorphism $A \to A/I$ under the functor $M \mapsto \tilde{M}$ from graded $A$-modules to quasi-coherent sheaves on $\mathrm{Proj}\, A$. Since this functor is exact {% stacks 01M7 %}, we have $\tilde{I} = \ker(i^\sharp)$. In other words, $\tilde{I}$ is the ideal sheaf of $V_+(I)$. 

Choose a list $f = (f_0, f_1, \dotsc, f_n)$ of elements of $A_1$ which generate $A$ as an algebra over $A_0$. Then  $f$ generates the irrelevant ideal $A_+$ {% stacks 07Z4 %}, so $D_+(f)$ is an affine open cover of $\mathrm{Proj}\, A$ {% stacks 00JP %}. Note that $D_+(f_i) = \mathrm{Spec}\, A_{(f_i)}$ and the composite $$D_+(f_i) \hookrightarrow \mathrm{Proj}\, A \to \mathrm{Spec}\, A_0$$ corresponds to the natural map $A_0 \to A_{(f_i)}$. We have $\Gamma(D_+(f_i), \tilde{I}) = I_{(f_i)}$. 

Suppose $x \in I \cap A_0$. We would like to show that $x \in I \cap A_0$ if and only if $x \in I_{(f_i)}$ for all $i$. The "only if" direction is clear, so suppose for the converse that $x \in I_{(f_i)}$ for all $i$. Then $x = y_i/f_i^{m_i}$ for some homogeneous $y_i \in I$ with $\deg(y_i) = m_i \deg(f_i) = m_i$. Letting $m = \max m_i$, we can replace $y_i$ with $y_i f_i^{m - m_i}$ in order to assume that $m = m_i$ for all $i$. In other words, we have $x = y_i / f_i^m$ for all $i$, which means that $f_i^m x \in I$ for all $i$. 

Suppose $r \geq (n+1)m$. Then, for any multi-index $\alpha = (\alpha_0, \dotsc, \alpha_n)$ such that $|\alpha| = r$, there must exist an $i$ such that $\alpha_i \geq m$. Then $f^\alpha$ is divisible by $f_i^m$, so $f^\alpha x \in I$. Thus $xA_r \subseteq I$. Since $I$ is saturated, we conclude that $x \in I$. 
<span style="float: right;">$\Box$</span>

# Projective Extension Theorem

Let $k$ be an arbitrary field. The "projective extension theorem" {% cite clo --label theorem --locator 8.5.6 %} is an immediate application of the result above in the special case where $$ A = k[t_1, \dotsc, t_m, x_0, x_1, \dotsc, x_n], $$
graded so that $\deg(t_1) = \cdots = \deg(t_m) = 0$ and $\deg(x_0) = \cdots = \deg(x_n) = 1$. In this case, the structure map $\pi : \mathrm{Proj}\,A \to \mathrm{Spec}\,A_0$ is the projection map $\mathbb{P}^n \times \mathbb{A}^m \to \mathbb{A}^m$. 

The one thing worth remarking on here is that the description of the scheme-theoretic image is amenable to computation in this case: {% cite clo --label theorem --locator 4.4.13--14 %} allows us to compute Gröbner basis for the saturation $I^s = (I : A_+^\infty)$ of $I$. Then $I^s \cap A_0$ is the $m$th elimination ideal of $I^s$, so we can compute a Gröbner basis for it using the elimination theorem {% cite clo --label theorem --locator 3.1.2 %}. 

# Affine Extension and Closure Theorems

We now specialize further to give a scheme-theoretic formulation of the "(affine) extension theorem" {% cite clo --label theorem --locator 3.1.3, 3.2.3 %} and the "closure theorem" {% cite clo --label theorem --locator 3.2.3, 4.4.4, 4.7.1 %}. Let $k$ be an arbitrary field. Let $t = (t_1, \dotsc, t_m)$ and $x$ be indeterminates. Let $\mathbb{A}^m = \mathrm{Spec}\, k[t]$ and $\mathbb{A}^1 \times \mathbb{A}^m = \mathrm{Spec}\, k[t,x]$. The inclusion $k[t] \to k[t,x]$ induces the projection 
$$ \pi : \mathbb{A}^1 \times \mathbb{A}^m \to \mathbb{A}^m. $$
Let $I \subseteq k[t,x]$ be an ideal, so that $V(I)$ is a closed subscheme of $\mathbb{A}^1 \times \mathbb{A}^m$. We would like to give a description of the set-theoretic image $\pi(V(I))$.

**Definition.** For any $f \in k[t,x]$, let $c_f \in k[t]$ denote the leading coefficient of $f$ when $f$ is regarded as a polynomial in $x$ with coefficients in $k[t]$.

**Proposition 3.** Suppose $I \subseteq k[t,x]$ is an ideal. Then $V(I \cap k[t])$ is the scheme-theoretic image of $V(I)$ under $\pi$, and as sets, we have 
$$ \pi(V(I)) \supseteq V(I \cap k[t]) \setminus V(c_{f_1}, \dotsc, c_{f_t}) $$
whenever $f_1, \dotsc, f_r \in I$. 

*Proof.* Since $(\pi^\sharp)^{-1}(I) = I \cap k[t]$, it is clear that $V(I \cap k[t])$ is the scheme-theoretic image of $V(I)$ under $\pi$. It is the characterization of the set-theoretic image that requires a bit of work. 

Let $\mathbb{P}^1 \times \mathbb{A}^m = \mathrm{Proj} \, k[t, x, y]$, where $k[t,x,y]$ is graded so that $\deg(t) = 0$ and $\deg(x) = \deg(y) = 1$. Observe that $\pi = \bar{\pi} \circ i$, where $i : \mathbb{A}^1 \times \mathbb{A}^m \to \mathbb{P}^1 \times \mathbb{A}^m$ is the open embedding and $\bar{\pi} : \mathbb{P}^1 \times \mathbb{A}^m \to \mathbb{A}^m$ is the projection. Let $I^h$ be the homogenization of $I$. Then $V_+(I^h)$ is the closure of $V(I)$ in $\mathbb{P}^1 \times \mathbb{A}^m$, ie, the scheme-theoretic image of $V(I)$ under $i$. Moreover, we have
$$ V_+(I^h) = i(V(I)) \sqcup V_+(J), $$
where $J = I^h + \langle y \rangle$ is the ideal defining the "points at infinity" of $V_+(I^h)$. By proposition 2 above and lemma 4 below, we have $$ \begin{aligned} \bar{\pi}(V_+(I^h)) &= V(I^{h,s} \cap k[t]) = V(I \cap k[t]) \\ \bar{\pi}(V_+(J)) &= V(J^s \cap k[t]). \end{aligned}$$
Thus we see that  
$$ \pi(V(I)) \cup V(J^s \cap k[t]) = V(I \cap k[tx]), $$ so it suffices to show that $V(J^s \cap k[t]) \subseteq  V(c_{f_1}, \dotsc, c_{f_r})$, or, in other words, that $c_{f_i} \in J^s \cap k[t]$ for all $i = 1, \dotsc, r$.

Let us show that $c_f \in J^s$ whenever $f \in I$. Observe that 
$$ f = c_f x^d + c_1 x^{d-1} + \cdots + c_d $$ 
for some $c_1, \dotsc, c_d \in k[t]$, and  
$$ f^h = c_f x^d + c_1 x^{d-1} y + \cdots + c_d y^d \in I^h \subseteq J. $$
Since $y \in J$ also, we see that $c_f x^d \in J$. Any monomial of degree $d$ containing $y$ is also in $J$, so $c_f \cdot k[t,x,y]_d \subseteq J$. Thus $c_f \in J^s$ by definition of saturation. 
<span style="float: right;">$\Box$</span>

**Lemma 4.** Let $f_1 : X \to Y_1$ and $g : Y_1 \to Y_2$ be morphisms of schemes. Set $f_2 = g \circ f_1 : X \to Y_2$, and let $Z_i \subseteq Y_i$ be the scheme-theoretic image of $f_i$ for $i = 1, 2$. Then $Z_2$ is the scheme-theoretic image of $Z_1$ under $g$. 

*Proof.* This is obvious.[^qc] Let $Z_2'$ be the scheme-theoretic image of $Z_1$ under $g$. Then $Z_2'$ is a closed subscheme of $Z_2$ by {% stacks 01R9 %}. Conversely, since $g|_{Z_1}$ factors through $Z_2'$, we see that $g \circ f_1 = f_2$ also factors through $Z_2'$. Thus $Z_2$ is a closed subscheme of $Z_2'$. 
<span style="float: right;">$\Box$</span>

[^qc]: Despite the obviousness, it's worth noting perhaps that this result is true without quasi-compactness assumptions on the morphisms (which is usually required for well-behavedness of scheme-theoretic images).

Proposition 3 is a direct scheme-theoretic translation of the affine extension theorem as well as the first part of the closure theorem {% cite clo --label theorem --locator 3.2.3(a), 4.4.4 %}. That being said, note that the statement furnishes the sharpest possible lower bound on $\pi(V(I))$ when $V(c_{f_1}, \dotsc, c_{f_t})$ is as small as possible, which happens when $\langle c_{f_1}, \dotsc, c_{f_r} \rangle$ is as large as possible. If $J = I^h + \langle y \rangle \subseteq k[t,x,y]$ is as in the proof above, we know that $\langle c_{f_1}, \dotsc, c_{f_r} \rangle \subseteq J^s \cap k[t]$, which leads to two questions: 

* Are there $f_1, \dotsc, f_r \in I$ such that $\langle c_{f_1}, \dotsc, c_{f_r} \rangle = J^s \cap k[t]$?

* If $\langle c_{f_1}, \dotsc, c_{f_r} \rangle = J^s \cap k[t]$, is it true that $\pi(V(I)) = V(I \cap k[t]) \setminus V(c_{f_1}, \dotsc, c_{f_r})$?

Bad news first: the second question has a negative answer, and in fact one does not need to look too hard to find counterexamples. While Chevalley's theorem {% stacks 00FE %} guarantees that $\pi(V(I))$ must be constructible, there is no reason that it must be locally closed. The simplest example of a constructible set which is not locally closed is the set 
$$S = V(u,t) \cup D(u) \subseteq \mathrm{Spec}\, k[t,u] = \mathbb{A}^2, $$
ie, $S$ is obtained from $\mathbb{A}^2$ by removing the $t$-axis except for the origin. Observe that $S$ is the set-theoretic image of a closed subscheme $Z \subseteq \mathbb{A}^3 = \mathrm{Spec}\, k[t,u,x]$ under the projection $\pi : \mathbb{A}^3 \to \mathbb{A}^2$. For example, it is easy to verify that $\pi(Z) = S$ for $$ Z = V(ux-1) \cup V(u, t). $$

But back to the good news: the first question has an affirmative answer, and in fact there is an algorithm for finding such polynomials! We simply let $G = \{f_1, \dotsc, f_r\}$ be a Gröbner basis for $I$ with respect to lex order where $x > t_1 > \cdots > t_m$.[^1] Let us show that this works.  

[^1]: We use this lex order for simplicity, but more computationally efficient orders can be chosen instead. For example, one might use the order where we first compare the degree of $x$, and then use grlex or grevlex to break ties. 

*Proof.* Observe that $G^h = \{f_1^h, \dotsc, f_r^h\}$ is a Gröbner basis for $I^h$ with respect to lex order where $x > y > t_1 > \cdots > t_m$ {% cite clo --label proposition --locator 8.5.10 %}, so $\tilde{H} = \{f_1^h, \dotsc, f_r^h, y\}$ generates $J$. We use a tilde here because we will shortly adjust this generating set and make it simpler, but first, we note that $\tilde{H}$ is actually a Gröbner basis for $J$. Since $G^h$ is Gröbner basis, we have $S(f_i^h, f_j^h) \to_{G^h} 0$ for all $i \neq j$, so $S(f_i^h, f_j^h) \to_{\tilde{H}} 0$ as well since $G^h \subseteq \tilde{H}$. Moreover, the leading monomial of $f_i^h$ does not contain $y$, so $S(f_i^h, y) \to_{\tilde{H}} 0$ for all $i$ {% cite clo --label proposition --locator 2.9.4 %}. Thus $\tilde{H}$ is a Gröbner basis for $J$ by Buchberger's criterion {% cite clo --label theorem --locator 2.9.3 %}.

The leading term of $f_i^h$ is $c_{f_i}x^{d_i}$ for some $d_i \in \N$ (ie, it is the same as the leading term of $f_i$). All other terms in $f_i^h$ contain $y$,  which is already in $J$. Thus $$H = \{c_{f_1}x^{d_1}, \cdots, c_{f_r}x^{d_r}, y\}$$ also generates $J$. Moreover, since $\tilde{H}$ is a Gröbner basis for $J$ and the leading terms of elements of $H$ are the same as those of $\tilde{H}$, we see that $H$ is also a Gröbner basis for $J$. 

Let $H^s = \{c_{f_1}, \dotsc, c_{f_r}, y\}$. This is evidently a subset of the saturation $J^s$, and I claim that $H^s$ is actually a Gröbner basis for $J^s$. In other words, I will show that, for any $g \in J^s$, the leading monomial $\mathrm{LM}(g)$ is divisible either by $y$ or by $\mathrm{LM}(c_{f_i})$ for some $i$ {% cite clo --label definition --locator 2.5.5 %}. Assume that it is not divisible by $y$. Since $g \in J^s$, we have $gx^d \in J$ for some $d$. We may increase $d$ without changing the fact that $gx^d \in J$, so let's increase $d$ until it is  larger than $\max\{d_1, \dotsc, d_r\}$. Then
$$ \mathrm{LM}(g)x^d = \mathrm{LM}(gx^d) \in \langle \mathrm{LM}(J) \rangle = \langle \mathrm{LM}(H) \rangle. $$
Since $\mathrm{LM}(g)x^d$ is not divisible by $y$, this means that there must exist some $i$ such that $\mathrm{LM}(g)x^d$ is divisible by $\mathrm{LM}(c_{f_i}x^{d_i}) = \mathrm{LM}(c_{f_i})x^{d_i}$. This means that $\mathrm{LM}(g)x^{d - d_i}$ is divisible by $\mathrm{LM}(c_{f_i})$. But $\mathrm{LM}(c_{f_i})$ is a monomial in $t_1, \dotsc, t_m$, so in fact this means that $\mathrm{LM}(g)$ is divisible by $\mathrm{LM}(c_{f_i})$, as desired.  

Finally, note that $H^s \cap k[t] = \{ c_{f_1}, \dotsc, c_{f_r} \}$ and that the elimination theorem implies that this is a Gröbner basis for $J^s \cap k[t]$. 
<span style="float: right;">$\Box$</span>

The upshot is the following. If we choose $G = \{f_1, \dotsc, f_r\}$ to be a Gröbner basis for $I$ as indicated above, then $I \cap k[t]$ is generated by $G \cap k[t]$ by the elimination theorem, and the constructible set $\pi(V(I))$ is sandwiched between the closed set $V(G \cap k[t])$ and the locally closed set $V(G \cap k[t]) \setminus V(c_{f_1}, \dotsc, c_{f_r})$. 
$$ V(G \cap k[t]) \supseteq \pi(V(I)) \supseteq V(G \cap k[t]) \setminus V(c_{f_1}, \dotsc, c_{f_r}) $$
In fact, $V(G \cap k[t]) \setminus  V(c_{f_1}, \dotsc, c_{f_r})$ is a maximal locally closed subset of $\pi(V(I))$. Finally, $G \cap k[t]$ and $c_{f_1}, \dotsc, c_{f_r}$ are both guaranteed to be Gröbner bases for the ideals they generate. Note moreover that $W = V(c_{f_1}, \dotsc, c_{f_r})$ satisfies the conclusion of the second part of the closure theorem {% cite clo --label theorem --locator 3.2.3(b), 4.7.1 %}

In short, we have an algorithmic way of finding both the closure as well as a maximal locally closed subset of the set-theoretic image of $V(I)$ under $\pi$.
