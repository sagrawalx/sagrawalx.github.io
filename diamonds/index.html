The Diamonds Project

The Diamonds Project is an attempt to consolidate foundations for the theory of adic spaces, perfectoid spaces, diamonds, etc. The hope is that this will one day be a fairly comprehensive reference about such material, much like The Stacks Project is vis-à-vis schemes, algebraic spaces, algebraic stacks, etc.

I'm hoping for this to be a collaborative endeavor; if you'd like to contribute, let's talk! Also, if you spot an error, please point it out!
